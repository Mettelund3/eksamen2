Javascript Burger King Project

2. Semester Assignment

Make the same amount of javascript things on an already existing website as the amount of group members.

The javascript pieces we make on the website has to contain all of our pensum of javascript. We chose Burger King and made some fun pieces around their site.

We made as close a copy as we could in a short amount of time of Burger Kings website. We didnt use a lot of time doing that because it wasn't our main focus, so i did it mainly by myself out of interest. We chose 5 different javascript thing to add to the site and we chose a Calender, Slideshow, Memory game, Quiz. I personally did a fun Memory Game, with the outside of the cards being Burger King Logo and on the other side it was different pictures from the menu.

When the memorygame is finished, you'll get an alert where it says that you won a free whopper. The memorygame is random everytime you use the site or update the site, so it doesn't remember the placement or that you maybe have won before.
